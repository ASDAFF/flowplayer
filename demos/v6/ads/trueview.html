ad_type

ad_type is "skippablevideo" and it is positioned at time zero, which makes it a pre-roll.

ad_type defines what types of ads will be requested and shown. Here we only show skippablevideo adverts. If you don't specify the ad_type value at all it will show video, image and text ads because the default ad_type value is "video_image_text".
