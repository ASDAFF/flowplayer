Player container and ads configuration

The player container with the ASF configuration in an inline SCRIPT tag:

ad_type defines what types of ads will be requested and shown. Here we only show image and text adverts. If you don't specify the ad_type value at all it will show video, image and text ads because the default ad_type value is "video_image_text".
